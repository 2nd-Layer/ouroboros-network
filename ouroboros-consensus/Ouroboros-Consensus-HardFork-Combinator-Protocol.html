type HardForkCanBeLeader xs = OptNP False WrapCanBeLeader xs Source #

CanBeLeader instance for HardForkProtocol

We allow an optional CanBeLeader proof per era, to make it possible to configure a node to, say, be able to produce Shelley but not Byron blocks. However, we do not allow all eras to be empty (since, in that case, we cannot be a leader).
