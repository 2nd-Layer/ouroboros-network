class MonadTrans (t ∷ (Type → Type) → Type → Type) where #

The class of monad transformers. Instances should satisfy the following laws, which state that lift is a monad transformation:

  • lift . return = return
  • lift (m >>= f) = lift m >>= (lift . f)
