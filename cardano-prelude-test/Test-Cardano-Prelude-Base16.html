Test.Cardano.Prelude.Base16

Description

Functions to encode and decode ByteStrings in base 16 with line wrapping

This encoding is used for producing pretty output from golden tests. The golden values are stored on disk as indexed, line-wrapped hex dumps. These can then be nicely diffed to provide some indication of where serialisation errors may have occurred.
