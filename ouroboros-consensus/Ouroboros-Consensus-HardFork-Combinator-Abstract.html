singleEraParams ∷ proxy blk → PartialLedgerConfig blk → EraParams Source #

Era parameters

The era parameters must be static (cannot depend on the ledger state).

Since we ne need this to construct the HardForkSummary (and hence the EpochInfo, this takes the partial config, not the full config (or we'd end up in a catch-22).

singleEraTransition ∷ PartialLedgerConfig blk → LedgerState blk → Maybe EpochNo Source #

Era transition

This should only report the transition point once it is stable (rollback cannot affect it anymore).

This takes the partial config rather than the full config for the same reason as singleEraParam.
