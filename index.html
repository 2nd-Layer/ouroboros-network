This site contains Haskell documentation of

  • Ouroboros-Network - networking component of the Haskell implementation of Ouroboros protocols
  • Ouroboros-Conensus - Ouroboros Conensus family of protocol which integrate with `Ouroboros-Network`

Module Structure And Network Components

  • Network - low level networking libraries, independent of our choices

    • Win32-network
    • typed-protocols
    • network-mux

  • Ouroboros.Network

    • ouroboros-network-framework - our instanciations and implementation of servers, clients, abstractions for working with named pipes and unix sockets. These are higher level components, which are independent of ouroboros protocols, but build a solid foundation for the networking library

    • ouroboros-network - networking library which supports ouroborso family of protocols. It includes

      • Implementations, type level sepcifications, and api of all the protocols:

        • ChainSync
        • BlockFetch
        • TxSubmission
        • LocalTxSubmission
        • LocalStateQuery
        • LocalTxMonitor
        • Handshake

      • Block chain related data structures that the networking code needs to be aware:

        • Block
        • ChainFramgent
        • AnchoredFragment

      • Network applications:

        • Block Fetch Application

        • Tx Submission Applications:

          • inbound
          • outbound
        • Peer Selection
        • Data Diffusion which combines all the pieces for supporting a node
        • NodeToNode API
        • NodeToClient API
