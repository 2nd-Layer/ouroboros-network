demux ∷ (MonadSTM m, MonadThrow m, MonadThrow (STM m), HasCallStack) ⇒ DemuxState m → m void Source #

demux runs as a single separate thread and reads complete MuxSDUs from the underlying Mux Bearer and forwards it to the matching ingress queue.

data DemuxState m Source #

Each peer's multiplexer has some state that provides both de-multiplexing details (for despatch of incoming mesages to mini protocols) and for dispatching incoming SDUs. This is shared between the muxIngress and the bearerIngress processes.
